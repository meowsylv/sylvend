Management!

Welcome to the management section! This place can only be accessed by the users listed in the users field in /etc/sylvend/secret.json

All users have the following syntax:

{
    "username": "sylv",
    "password": "Nh5I0DCPIOMtul+1Yyi68Y1y7wzLQ7hPXCYtKmofxsg=" //SHA256 hash of this user's password.
}

Another password-protected endpoint is /api/management. This is an api endpoint used to modify sylvend's configuration.

sylvend does not come with a web management UI by default, you have to make it yourself. Or you can just use the one I made for my website.
